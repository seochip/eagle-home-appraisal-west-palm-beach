WordPress 6.4.2

Thank you for choosing WordPress. This readme file contains basic installation and usage instructions.

System Requirements

  • PHP version 7.4 or greater.
  • MySQL version 5.7 OR MariaDB version 10.3 or greater.
  • HTTPS support

Installation

  1. Unzip the package in an empty directory and upload everything.
  2. Open wp-admin/install.php in your browser.
  3. Fill in your database connection details.
